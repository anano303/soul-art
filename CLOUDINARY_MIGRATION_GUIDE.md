# Cloudinary Migration Guide

## ­ЪЊІ рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

рЃћрЃА рЃњрЃљрЃўрЃЊрЃў рЃљрЃдрЃгрЃћрЃарЃА рЃарЃЮрЃњрЃЮрЃа рЃЏрЃЮрЃЋрЃљрЃ«рЃЊрЃўрЃюрЃЮрЃЌ рЃДрЃЋрЃћрЃџрЃљ рЃАрЃБрЃарЃљрЃЌрЃўрЃА, рЃЋрЃўрЃЊрЃћрЃЮрЃАрЃљ рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃљрЃАрЃћрЃЌрЃћрЃЉрЃўрЃА рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃљ рЃФрЃЋрЃћрЃџрЃў Cloudinary рЃћрЃЦрЃљрЃБрЃюрЃбрЃўрЃЊрЃљрЃю (`dsufx8uzd`) рЃљрЃ«рЃљрЃџ рЃћрЃЦрЃљрЃБрЃюрЃбрЃерЃў (`dwfqjtdu2`).

## ­ЪћЇ рЃарЃљ рЃљрЃарЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃБрЃџрЃў

рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃўрЃА рЃАрЃЎрЃарЃўрЃърЃбрЃћрЃЉрЃў рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃћрЃю рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЎрЃЮрЃџрЃћрЃЦрЃфрЃўрЃћрЃЉрЃА MongoDB-рЃерЃў:

- **Products** - `images` рЃЊрЃљ `thumbnail` рЃЋрЃћрЃџрЃћрЃЉрЃў
- **Users** - `profilePicture` рЃЋрЃћрЃџрЃў
- **Banners** - `imageUrl` рЃЋрЃћрЃџрЃў
- **Blog Posts** - `image` рЃЋрЃћрЃџрЃў
- **Categories** - `image` рЃЋрЃћрЃџрЃў
- **.env рЃцрЃљрЃўрЃџрЃў** - `SLIDESHOW_OUTRO_IMAGE_URL` рЃЊрЃљ `SLIDESHOW_AUDIO_URL`

## ­ЪЊі рЃАрЃбрЃљрЃбрЃўрЃАрЃбрЃўрЃЎрЃљ

```bash
npm run cloudinary:check
```

рЃћрЃА рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ рЃњрЃћрЃбрЃДрЃЋрЃўрЃЌ:

- рЃарЃљрЃЏрЃЊрЃћрЃюрЃў URL рЃБрЃюрЃЊрЃљ рЃЏрЃўрЃњрЃарЃўрЃарЃЊрЃћрЃА рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџ рЃЎрЃЮрЃџрЃћрЃЦрЃфрЃўрЃљрЃерЃў
- рЃарЃљрЃЏрЃЊрЃћрЃюрЃў URL рЃБрЃЎрЃЋрЃћ рЃњрЃљрЃЊрЃљрЃбрЃљрЃюрЃўрЃџрЃўрЃљ рЃљрЃ«рЃљрЃџ рЃћрЃЦрЃљрЃБрЃюрЃбрЃерЃў
- рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃА рЃФрЃЋрЃћрЃџрЃў URL-рЃћрЃЉрЃўрЃЊрЃљрЃю

## ­Ъџђ рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў

### рЃюрЃљрЃЉрЃўрЃ»рЃў 1: Dry Run - рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ

рЃ»рЃћрЃа рЃњрЃљрЃБрЃерЃЋрЃў dry-run рЃарЃћрЃЪрЃўрЃЏрЃерЃў рЃарЃљрЃЌрЃљ рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃћ рЃарЃЮрЃЏ рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў рЃАрЃгрЃЮрЃарЃљрЃЊ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА:

#### Database рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃљ (Dry Run)

```bash
npm run cloudinary:migrate:dry-run
```

#### .env рЃцрЃљрЃўрЃџрЃўрЃА рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃљ (Dry Run)

```bash
npm run cloudinary:migrate:env:dry-run
```

Dry-run рЃарЃћрЃЪрЃўрЃЏрЃерЃў:

- рЃљрЃарЃљрЃцрЃћрЃарЃў рЃерЃћрЃўрЃфрЃЋрЃџрЃћрЃЉрЃљ
- рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃфрЃюрЃЮрЃЉрЃћрЃЉрЃљ рЃарЃљрЃА рЃњрЃљрЃљрЃЎрЃћрЃЌрЃћрЃЉрЃЊрЃљ рЃАрЃЎрЃарЃўрЃърЃбрЃў
- рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃЮ рЃарЃЮрЃЏ рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў рЃАрЃгрЃЮрЃарЃўрЃљ

### рЃюрЃљрЃЉрЃўрЃ»рЃў 2: .env рЃцрЃљрЃўрЃџрЃўрЃА рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃљ

рЃ»рЃћрЃа рЃЊрЃљрЃЋрЃўрЃгрЃДрЃЮрЃЌ .env рЃцрЃљрЃўрЃџрЃўрЃА рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃўрЃЌ (рЃарЃљрЃЊрЃњрЃљрЃю рЃћрЃА рЃБрЃцрЃарЃЮ рЃАрЃгрЃарЃљрЃцрЃўрЃљ):

```bash
npm run cloudinary:migrate:env
```

рЃћрЃА рЃАрЃЎрЃарЃўрЃърЃбрЃў:

1. рЃФрЃЋрЃћрЃџрЃў рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃў/рЃЋрЃўрЃЊрЃћрЃЮрЃћрЃЉрЃў рЃњрЃљрЃЊрЃЏрЃЮрЃўрЃгрЃћрЃарЃА
2. рЃљрЃбрЃЋрЃўрЃарЃЌрЃљрЃЋрЃА рЃљрЃ«рЃљрЃџ Cloudinary рЃћрЃЦрЃљрЃБрЃюрЃбрЃерЃў
3. рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃюрЃљрЃљрЃ«рЃџрЃћрЃЉрЃА .env рЃцрЃљрЃўрЃџрЃА рЃљрЃ«рЃљрЃџрЃў URL-рЃћрЃЉрЃўрЃЌ

### рЃюрЃљрЃЉрЃўрЃ»рЃў 3: Database рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃљ

рЃљрЃ«рЃџрЃљ рЃЊрЃљрЃЋрЃўрЃгрЃДрЃЮрЃЌ MongoDB-рЃўрЃА рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃљ:

```bash
npm run cloudinary:migrate
```

Рџа№ИЈ **рЃњрЃљрЃцрЃарЃЌрЃ«рЃўрЃџрЃћрЃЉрЃљ**: рЃћрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЊрЃўрЃЊрЃ«рЃљрЃюрЃА рЃњрЃљрЃњрЃарЃФрЃћрЃџрЃЊрЃћрЃА (361 рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃў + 3 рЃЉрЃљрЃюрЃћрЃарЃў = 364 URL)

рЃАрЃЎрЃарЃўрЃърЃбрЃў:

1. рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃўрЃА рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃА рЃњрЃљрЃЊрЃЏрЃЮрЃгрЃћрЃарЃА
2. рЃљрЃбрЃЋрЃўрЃарЃЌрЃљрЃЋрЃА рЃљрЃ«рЃљрЃџ Cloudinary рЃћрЃЦрЃљрЃБрЃюрЃбрЃерЃў
3. рЃњрЃљрЃюрЃљрЃљрЃ«рЃџрЃћрЃЉрЃА MongoDB рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА рЃљрЃ«рЃљрЃџрЃў URL-рЃћрЃЉрЃўрЃЌ
4. рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃЊрЃћрЃбрЃљрЃџрЃБрЃа рЃърЃарЃЮрЃњрЃарЃћрЃАрЃА

### рЃюрЃљрЃЉрЃўрЃ»рЃў 4: рЃњрЃљрЃЊрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ

рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃњрЃљрЃБрЃерЃЋрЃў рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ рЃ«рЃћрЃџрЃљрЃ«рЃџрЃљ:

```bash

```

рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃўрЃюрЃљрЃ«рЃЮ рЃарЃЮрЃЏ:

## РџЎ№ИЈ рЃАрЃЎрЃарЃўрЃърЃбрЃћрЃЉрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљ

### `check-cloudinary-urls.ts`

- MongoDB-рЃерЃў рЃФрЃЋрЃћрЃџрЃў Cloudinary URL-рЃћрЃЉрЃўрЃА рЃљрЃдрЃЏрЃЮрЃЕрЃћрЃюрЃљ
- рЃАрЃбрЃљрЃбрЃўрЃАрЃбрЃўрЃЎрЃўрЃА рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљ
- рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃўрЃА рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљ

### `migrate-cloudinary.ts`

- рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃўрЃА рЃАрЃЎрЃарЃўрЃърЃбрЃў MongoDB-рЃАрЃЌрЃЋрЃўрЃА
- рЃњрЃљрЃЊрЃЏрЃЮрЃгрЃћрЃарЃА рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃА рЃФрЃЋрЃћрЃџрЃў рЃћрЃЦрЃљрЃБрЃюрЃбрЃўрЃЊрЃљрЃю
- рЃљрЃбрЃЋрЃўрЃарЃЌрЃљрЃЋрЃА рЃљрЃ«рЃљрЃџ рЃћрЃЦрЃљрЃБрЃюрЃбрЃерЃў
- рЃњрЃљрЃюрЃљрЃљрЃ«рЃџрЃћрЃЉрЃА рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА

### `migrate-env-cloudinary.ts`

- .env рЃцрЃљрЃўрЃџрЃўрЃА рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃљ
- рЃљрЃЉрЃюрЃћрЃюрЃћрЃЉрЃА рЃЊрЃљ рЃљрЃбрЃЋрЃўрЃарЃЌрЃљрЃЋрЃА рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃА/рЃЋрЃўрЃЊрЃћрЃЮрЃћрЃЉрЃА
- рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃюрЃљрЃљрЃ«рЃџрЃћрЃЉрЃА .env рЃцрЃљрЃўрЃџрЃА

## ­ЪЊЂ Folder рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ Cloudinary-рЃерЃў

рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃљрЃАрЃћрЃЌрЃћрЃЉрЃў рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњ folder-рЃћрЃЉрЃерЃў:

- `ecommerce/` - рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃўрЃА рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃў
- `users/` - рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃцрЃўрЃџрЃўрЃА рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃў
- `banners/` - рЃЉрЃљрЃюрЃћрЃарЃўрЃА рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃў
- `blog/` - рЃЉрЃџрЃЮрЃњрЃўрЃА рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃў
- `categories/` - рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃў
- `images/` - .env-рЃўрЃА рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃў
- `audio/` - .env-рЃўрЃА рЃљрЃБрЃЊрЃўрЃЮ рЃцрЃљрЃўрЃџрЃћрЃЉрЃў

## Рџа№ИЈ рЃњрЃљрЃцрЃарЃЌрЃ«рЃўрЃџрЃћрЃЉрЃћрЃЉрЃў

1. **Backup**: рЃњрЃљрЃЊрЃљрЃЊрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА backup-рЃўрЃА рЃњрЃљрЃЎрЃћрЃЌрЃћрЃЉрЃљ рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃљрЃЏрЃЊрЃћ
2. **рЃЊрЃарЃЮ**: рЃЏрЃЌрЃћрЃџрЃў рЃърЃарЃЮрЃфрЃћрЃАрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃАрЃљрЃљрЃЌрЃў рЃњрЃљрЃњрЃарЃФрЃћрЃџрЃЊрЃћрЃА
3. **API Limits**: Cloudinary-рЃА рЃљрЃЦрЃЋрЃА API rate limits
4. **рЃЦрЃАрЃћрЃџрЃў**: рЃАрЃбрЃљрЃЉрЃўрЃџрЃБрЃарЃў рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃб рЃЎрЃљрЃЋрЃерЃўрЃарЃў рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ
5. **рЃљрЃЊрЃњрЃўрЃџрЃў**: рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃў рЃарЃЮрЃЏ рЃљрЃ«рЃљрЃџ рЃћрЃЦрЃљрЃБрЃюрЃбрЃерЃў рЃАрЃљрЃЎрЃЏрЃљрЃарЃўрЃАрЃў storage рЃљрЃарЃўрЃА

## ­Ъћё рЃЌрЃБ рЃарЃљрЃўрЃЏрЃћ рЃљрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА

## ­ЪЊѕ рЃърЃарЃЮрЃњрЃарЃћрЃАрЃўрЃА рЃЌрЃЋрЃљрЃџрЃДрЃБрЃарЃўрЃА рЃЊрЃћрЃЋрЃюрЃћрЃЉрЃљ
