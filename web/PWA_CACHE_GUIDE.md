# ­ЪЊ▒ PWA Cache Management - рЃљрЃ«рЃАрЃюрЃљ рЃЊрЃљ рЃњрЃљрЃЊрЃљрЃГрЃарЃўрЃА рЃњрЃќрЃћрЃЉрЃў

## РЮЊ рЃарЃљрЃбрЃЮрЃЏ рЃ«рЃЊрЃћрЃЉрЃљ рЃљрЃАрЃћ?

PWA (Progressive Web App) рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃљ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА **Cache Strategy**-рЃА рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃўрЃАрЃљ рЃЊрЃљ offline рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА:

### ­Ъћё Cache рЃАрЃбрЃарЃљрЃбрЃћрЃњрЃўрЃћрЃЉрЃў:

1. **NetworkFirst** - рЃърЃўрЃарЃЋрЃћрЃџ рЃарЃўрЃњрЃерЃў рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃбрЃўрЃЊрЃљрЃю, рЃерЃћрЃЏрЃЊрЃћрЃњ cache-рЃўрЃЊрЃљрЃю
2. **CacheFirst** - рЃърЃўрЃарЃЋрЃћрЃџ рЃарЃўрЃњрЃерЃў cache-рЃўрЃЊрЃљрЃю, рЃерЃћрЃЏрЃЊрЃћрЃњ рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃбрЃўрЃЊрЃљрЃю
3. **StaleWhileRevalidate** - cache-рЃўрЃЊрЃљрЃю + background-рЃерЃў рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ

## ­ЪЏа№ИЈ рЃЕрЃЋрЃћрЃюрЃў рЃљрЃЏрЃЮрЃ«рЃАрЃюрЃљ

### РюЁ рЃарЃљ рЃњрЃљрЃЋрЃљрЃЎрЃћрЃЌрЃћрЃЌ:

1. **рЃўрЃюрЃбрЃћрЃџрЃћрЃЦрЃбрЃБрЃљрЃџрЃБрЃарЃў Cache рЃАрЃбрЃарЃљрЃбрЃћрЃњрЃўрЃљ:**

   ```
   - API calls: 5 рЃгрЃБрЃЌрЃўрЃљрЃюрЃў cache
   - рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃў: 7 рЃЊрЃдрЃўрЃљрЃюрЃў cache
   - Static files: 7 рЃЊрЃдрЃўрЃљрЃюрЃў cache
   - Pages: 10 рЃгрЃБрЃЌрЃўрЃљрЃюрЃў cache
   ```

2. **рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў Cache рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ:**

   - Service Worker рЃДрЃЮрЃЋрЃћрЃџ 10 рЃгрЃБрЃЌрЃерЃў рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА рЃАрЃўрЃљрЃ«рЃџрЃћрЃА
   - API responses-рЃА timestamp рЃћрЃЏрЃљрЃбрЃћрЃЉрЃљ
   - рЃЏрЃЮрЃФрЃЋрЃћрЃџрЃћрЃЉрЃБрЃџрЃў cache рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃюрЃљрЃ«рЃџрЃЊрЃћрЃЉрЃљ

3. **рЃЏрЃљрЃюрЃБрЃљрЃџрЃБрЃарЃў Cache Control:**

   - ­Ъћё Cache рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃўрЃА рЃдрЃўрЃџрЃљрЃЎрЃў (development-рЃерЃў)
   - рЃюрЃЮрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃљ рЃљрЃ«рЃљрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ
   - рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃў Status Indicator

4. **Smart Cache Invalidation:**
   - API рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў - 5 рЃгрЃБрЃЌрЃў
   - Page content - 10 рЃгрЃБрЃЌрЃў
   - рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃћрЃЉрЃќрЃћ рЃљрЃЋрЃбрЃЮ-рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ

## ­ЪЊІ рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ:

### Development рЃарЃћрЃЪрЃўрЃЏрЃерЃў:

- ­Ъћё рЃдрЃўрЃџрЃљрЃЎрЃў рЃ«рЃўрЃџрЃЋрЃљрЃЊрЃўрЃљ рЃќрЃћрЃЊрЃљ рЃЏрЃљрЃарЃ»рЃЋрЃюрЃўрЃЋ
- рЃЊрЃљрЃГрЃћрЃарЃўрЃЌ - cache рЃАрЃарЃБрЃџрЃљрЃЊ рЃўрЃгрЃЏрЃўрЃюрЃЊрЃћрЃЉрЃљ
- рЃњрЃЋрЃћрЃарЃЊрЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃюрЃљрЃ«рЃџрЃЊрЃћрЃЉрЃљ

### Production-рЃерЃў:

- Cache рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃЏрЃљрЃарЃЌрЃБрЃџрЃўрЃљ
- рЃюрЃЮрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃў рЃљрЃ«рЃљрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ
- рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃљрЃа рЃАрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ manual intervention

## ­Ъј» рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА рЃарЃЕрЃћрЃЋрЃћрЃЉрЃў:

### рЃЌрЃБ рЃФрЃЋрЃћрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃЕрЃљрЃюрЃА:

1. **рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃЏрЃЮрЃџрЃЮрЃЊрЃўрЃюрЃў:** 5-10 рЃгрЃБрЃЌрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃњрЃљрЃюрЃљрЃ«рЃџрЃЊрЃћрЃЉрЃљ
2. **Manual refresh:** рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃўрЃА refresh (Ctrl+R)
3. **Hard refresh:** Ctrl+Shift+R (cache рЃњрЃЋрЃћрЃарЃЊрЃўрЃА)
4. **PWA restart:** App-рЃўрЃА рЃЊрЃљрЃ«рЃБрЃарЃЋрЃљ рЃЊрЃљ рЃ«рЃћрЃџрЃљрЃ«рЃџрЃљ рЃњрЃљрЃ«рЃАрЃюрЃљ

### PWA-рЃерЃў:

- App-рЃўрЃА рЃЊрЃљрЃ«рЃБрЃарЃЋрЃљ/рЃњрЃљрЃ«рЃАрЃюрЃљ рЃљрЃ«рЃљрЃџ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА рЃўрЃбрЃљрЃюрЃА
- Notification рЃЏрЃЮрЃЋрЃљ рЃљрЃ«рЃљрЃџрЃў info-рЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ
- Background-рЃерЃў рЃљрЃЋрЃбрЃЮ-рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА

## ­ЪћД рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃў рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃў:

### Cache Hierarchy:

```javascript
1. Network Check (3 рЃгрЃљрЃЏрЃў timeout)
2. Fresh Cache (рЃЌрЃБ < 5-10 рЃгрЃБрЃЌрЃў)
3. Stale Cache (backup)
4. Fallback Content
```

### Cache Categories:

- **api-cache:** API responses (5 рЃгрЃБрЃЌрЃў)
- **images-cache:** рЃюрЃљрЃ«рЃљрЃбрЃћрЃЉрЃў (7 рЃЊрЃдрЃћ)
- **static-cache:** CSS/JS files (7 рЃЊрЃдрЃћ)
- **pages-cache:** HTML pages (10 рЃгрЃБрЃЌрЃў)

### Auto-Update Triggers:

- Timer-based (рЃДрЃЮрЃЋрЃћрЃџ 10 рЃгрЃБрЃЌрЃерЃў)
- User action (page navigation)
- Push notification received
- Network reconnection

## ­Ъџђ рЃБрЃърЃўрЃарЃљрЃбрЃћрЃАрЃЮрЃЉрЃћрЃЉрЃў:

РюЁ **рЃњрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃБрЃџрЃў рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃљ** - cache-рЃЊрЃљрЃю рЃАрЃгрЃарЃљрЃцрЃў loading
РюЁ **Offline рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ** - рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃбрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃЮрЃЉрЃљ  
РюЁ **рЃЉрЃљрЃюрЃЊрЃЋрЃўрЃЊрЃбрЃўрЃА рЃћрЃЎрЃЮрЃюрЃЮрЃЏрЃўрЃљ** - рЃюрЃљрЃЎрЃџрЃћрЃЉрЃў рЃбрЃарЃљрЃцрЃўрЃЎрЃў
РюЁ **рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃќрЃћ рЃЏрЃЮрЃарЃњрЃћрЃЉрЃБрЃџрЃў** - рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ
РюЁ **рЃЏрЃЮрЃ«рЃћрЃарЃ«рЃћрЃЉрЃБрЃџрЃў Management** - visual feedback
